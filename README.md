# 💼 JobLens – AI Resume Job Predictor

**JobLens** is an AI-powered web application that helps users analyze their resumes, evaluate ATS compatibility, and discover personalized job recommendations. Built with **React** and **Flask**, JobLens simplifies the job-hunting process with intelligent insights.

## 🚀 Features

- 📄 **Resume Upload & Parsing**  
  Upload your resume (PDF/DOCX) and extract key details like skills, experience, and education.

- 🤖 **AI-Based Job Prediction**  
  Get job suggestions based on your resume content using NLP-powered matching.

- 📊 **ATS Compatibility Score**  
  Evaluate how well your resume performs in Applicant Tracking Systems.

- 🔍 **Skill Gap Insights**  
  Discover missing or in-demand skills for your desired roles.

- 🔗 **Job Application Links**  
  Get relevant job listings with direct application links.
